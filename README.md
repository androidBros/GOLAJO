# _GOLAJO_

#### 🐣차선호 <br>  🐣하동혁
#### 기간 : 2022/03/18 ~ 2022/04/22
#### OS : Android
<br>


### _프로젝트 상세내용_
***
### 주제

```sh
음식메뉴 선택 시 도움이 되는 랜덤음식 뽑기 
```
<br>
<br>
    

### 배경

```sh
음식메뉴 선택 시 결정을 못하는 사람들이 많습니다. 
저희 역시 음식메뉴를 결정하지 못한 적이 많았었기 때문에
음식메뉴를 선택해주는 프로그램이 있으면 좋을 것 같다는 생각을 하였습니다.
```
<br>
<br>

### 목적 및 용도

```sh
음식메뉴 선택 시 랜덤으로 메뉴가 선택되어
메뉴 선택에 대한 고민을 덜어줍니다.
```
<br>
<br>
